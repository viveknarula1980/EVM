*Index 1 is top of the stack. See [PUSH](/#60).*

## Notes

For out of bound bytes, 0s will be copied.

## Stack input

0. `mem_offset`: byte offset in the [memory](/about) where the result will be copied to.
1. `offset`: byte offset in the [data section](/about) to copy.
2. `size`: byte size to copy.

## Examples

| Data Section |
|---------|
| `0xFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF` |

| Memory before |

| Memory after |
|---------------------:|

## Error cases

The state changes done by the current context are [reverted](#FD) in those cases:
- Not enough gas.
- Execution in a legacy (non-EOF) context.
